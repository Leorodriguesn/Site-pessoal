Sigla para HyperText Markup Language, atualmente na versão HTML5, é uma linguagem que compõe a maior parte das páginas da internet e dos aplicativos online. Ele permite a construção de websites e a inserção de novos conteúdos, como textos, imagens e vídeos.

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo a um documento web. Atualmente na versão CSS3, sendo utilizada utilizada para otimizar o conteúdo das páginas e possibilitar uma apresentação mais amigável para o usuário.

Ao lado de HTML e CSS, é uma das três principais tecnologias da World Wide Web.
Javascript, ou simplesmente JS, é uma linguagem de programação de uso geral, aplicada principalmente para desenvolvimento web e desenvolvimento de software. Utilizado em programação Front-End, permite implementar efeitos dinâmicos complexos em páginas web.
